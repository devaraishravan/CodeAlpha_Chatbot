# CodeAlpha Chatbot

## Project Overview

CodeAlpha Chatbot is a web-based chatbot developed using HTML, CSS, and JavaScript. The chatbot provides instant responses to user queries using predefined input patterns and responses.

## Features

* Instant response to user messages
* User-friendly chat interface
* Predefined question-answer patterns
* Website integration
* Auto-scroll chat window
* Enter key support
* Responsive design

## Technologies Used

* HTML
* CSS
* JavaScript

## Supported Queries

* Hello
* Hi
* Services
* Price
* Contact
* Email
* Phone
* Internship
* Location
* About
* Help
* Working Hours
* Website
* Thank You
* Bye

## How to Run

1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Open index.html using Live Server.
4. Start chatting with the bot.
